# Review Checklists by Domain

## Machine Learning Papers

- [ ] Hyperparameter sensitivity analysis included?
- [ ] Compute budget reported (GPU hours, hardware)?
- [ ] Random seeds varied and results averaged?
- [ ] Comparison against recent SOTA, not just old baselines?
- [ ] Training/validation/test split clearly described?
- [ ] Code/data availability stated?

## Clinical / Biomedical Papers

- [ ] Study design appropriate (RCT, cohort, case-control)?
- [ ] Sample size justified with power analysis?
- [ ] IRB/ethics approval mentioned?
- [ ] Patient demographics reported?
- [ ] CONSORT/STROBE/PRISMA guidelines followed?
- [ ] Conflicts of interest declared?

## Systems / Engineering Papers

- [ ] Benchmarks are standard and reproducible?
- [ ] Scalability tested across different scales?
- [ ] Failure modes discussed?
- [ ] Comparison includes both latency and throughput?
- [ ] Real-world deployment evidence (not just synthetic)?
